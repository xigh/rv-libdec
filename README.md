# rv-dec

Very simple riscv opcodes decoder.

### Notes :

It compiles on Linux/FreeBSD, MacOS and Windows (both with WSL and VS 2019 Compiler).

### Requirements :

* CMake
* C compiler

### How to download :

```sh
git clone --recurse-submodules https://github.com/xigh/rv-libdec
cd rv-libdec
```

### How to compile :

```sh
mkdir build
cd build
cmake ..
cmake --build .
```
